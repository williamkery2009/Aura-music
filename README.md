# About Me

I'm **William Kery**, a student developer from India with a strong interest in frontend development, user interface design, and modern digital experiences.

I enjoy building visually refined interfaces, experimenting with creative concepts, and learning through practical projects. My approach to development focuses on clarity, responsiveness, and user experience rather than simply making things work.

Over time, I’ve developed a growing interest in:
- Frontend engineering
- UI/UX systems
- Android customization
- Interactive web experiences
- Modern product design

I believe technology becomes more meaningful when functionality and design work together seamlessly.
